# Project Roadmap - LifeMax

This roadmap highlights milestones, phases, and feature progression for LifeMax.

## Phase 1: Foundation (V1)
- [x] Initial database configuration & Neon migration.
- [x] Project architecture directory layout and Express bootstrap.
- [ ] User authentication (Registration, Login, JWT tokens).
- [ ] Core CRUD API endpoints (Categories, Activities, Time Blocks).
- [ ] Basic Analytics calculations (Effective Ratio formula).
- [ ] Habits and Goals tracking API.
- [ ] Front-end Dashboard interface integration.

## Phase 2: Refinement & Advanced Tracking (V2)
- [ ] Pomodoro Timer built-in.
- [ ] Google Calendar two-way synchronization.
- [ ] Enhanced journal & mood analytics.
- [ ] AI Insights based on productivity logs.
- [ ] Weekly/Monthly Review generators.
- [ ] Book tracking & Expense ledger integration.

## Future Plans
- [ ] High scalability migrations (AWS EC2 backend, AWS RDS database).
- [ ] Native mobile wrapper apps.
- [ ] Advanced CI/CD workflows using GitHub Actions.
